# Memory Match



## Technologies Used

- HTML5
- CSS3
- jQuery
- AWS EC2

## Live Demo

Try the application live at [https://hearthstone.davidjinhong.com](https://hearthstone.davidjinhong.com)

## Features

- User can win by matching all the card pairs.
- User can shuffle cards.
- User can reset the game.
